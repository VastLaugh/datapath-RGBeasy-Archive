<h1>datapath-RGBeasy</h1>
<p dir="auto">A video capture plugin for OBS Studio that uses the RGBEasy API</p>
<p dir="auto">To compile this, add it to a Visual Studio project or something and make sure you have the RGBEasy SDK and libobs somewhere in the path where your project looks for include files and libraries.</p>
